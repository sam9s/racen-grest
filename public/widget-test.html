That's it! Just one line of code adds the full RACEN chat experience to any page on your website. The widget will:

  • Display a floating chat bubble in the bottom-right corner
  • Open a chat window when clicked
  • Connect to RACEN for AI-powered responses
  • Stream responses in real-time
  • Support clickable links to JoveHeal programs
  • Work on mobile and desktop
